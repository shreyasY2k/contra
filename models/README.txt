This directory should contain the following GLTF models:

- soldier.glb - Player model
- enemy_soldier.glb - Enemy soldier model
- enemy_tank.glb - Enemy tank model
- enemy_helicopter.glb - Enemy helicopter model
- jeep.glb - Jeep vehicle model
- tank.glb - Tank vehicle model
- helicopter.glb - Helicopter vehicle model

If these files are not present, the game will use simple geometric models instead.

You can find free game-ready models at websites like:
- https://sketchfab.com (many free models)
- https://free3d.com
- https://www.turbosquid.com (has free models)
- https://www.mixamo.com (for character models and animations)
